Digit Bonus Ratios (Other Slots)

Bonus RTP% exceeds RTP FP%.

Payout Ratio is calculated as 1 + (baseRatio * baseCount + sum(bonusRatio_i * bonusWeight_i)) / (baseCount + sum(bonusWeight_i)), where baseCount = max(totalRoll - sum(bonusWeight_i), 0) and totalRoll references the slot's Total Counts input (or the global Bonus slot chance total when the per-slot value is empty).

Single Digit Payout & Bonus

Bonus RTP% exceeds RTP FP%.

Single BonusRTP uses all three suggest-win channels: SuggestWin%x1*(1+(baseRatio_x1*baseCount+sum(bonusRatio_i*bonusWeight_i))/(baseCount+sum(bonusWeight_i))) + SuggestWin%x2*(1+(baseRatio_x2*baseCount+sum(bonusRatio_i*bonusWeight_i))/(baseCount+sum(bonusWeight_i))) + SuggestWin%x3*(1+(baseRatio_x3*baseCount+sum(bonusRatio_i*bonusWeight_i))/(baseCount+sum(bonusWeight_i))) , where baseRatio_x1 is row payout ratio, and baseRatio_x2/x3 are the single multipliers.
